# AI-Native Roadmap

PortGuard is not meant to remain a static page. It should become a useful AI-era product that helps security analysts, IT admins, and MSP operators.

## Working v1 Intelligence

- Structured intelligence endpoint: `POST /api/intelligence`
- Deterministic product engine in `lib/product-engine.ts`
- UI calls the endpoint from the live command panel
- Output includes confidence, immediate outputs, automation plan, future add-ons, and contributor lanes

## Futuristic Add-Ons

- v2: Nmap XML import
- v2: cloud security group review
- v3: change-diff risk score
- v3: team remediation SLA board

## Contributor Challenge

Build the integrations and memory layer that make this product easier for the next generation to use. The best contributions should reduce manual effort, improve trust, and make the first useful result faster.
